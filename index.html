Final Project

Bachelor of Informatics Engineering

Development of SSTRANGE Web Application With Generative AI Integration for Code Plagiarism Detection
The increasing demand for efficient and accurate tools to manage code similarity has become crucial in higher education, particularly in computer science and programming. This study focuses on developing the SSTRANGE web application, which transitions from a desktop-based application to a web platform using Laravel. SSTRANGE integrates Generative AI to enhance functionality in detecting code plagiarism and generating code recommendations. The research employed both primary data, derived from academic submissions, and secondary data, collected from technological documentation. Various tests were conducted to evaluate system performance, user satisfaction, and the effectiveness of Generative AI integration, and response times during system testing ranged from 2.99 to 8.9 seconds, with user satisfaction achieving an average of 4 out of 5 on a Likert scale. The Generative AI accuracy test showed a result of 80.97%, while the uniqueness test indicated 80% originality in code recommendations. Furthermore, similarity detection testing on student submissions yielded an average similarity rate of 42.42%. These findings highlight SSTRANGE's capability to efficiently and reliably manage academic submissions, substantially improving speed, accuracy, and user experience. SSTRANGE significantly contributes to modernizing academic evaluation tools by promoting fairness, transparency, and efficiency in assessing academic works, particularly in programming and computer science education.

IEEE TALE International Conference 2023 (Co-Author)

Auckland, New Zealand

IEEE TALE International Conference 2023 Plagiarism and AI Assistance Misuse in Web Programming: Unfair Benefits and Characteristics

In programming education, plagiarism and misuse of artificial intelligence (AI) assistance are emerging issues. However, not many relevant studies are focused on web programming. We plan to develop automated tools to help instructors identify both misconducts. To fully understand the issues, we conducted a controlled experiment to observe the unfair benefits and the characteristics. We compared student performance in completing web programming tasks independently, with a submission to plagiarize, and with the help of AI assistance (ChatGPT). Our study shows that students who are involved in such misconducts get comparable test marks with less completion time. Plagiarized submissions are similar to the independent ones except in trivial aspects such as color and identifier names. AI-assisted submissions are more complex, making them less readable. Students believe AI assistance could be useful given proper acknowledgment of the use, although they are not convinced with readability and correctness of the solutions.

JAIC 2024 (Author)

Politeknik Negeri Batam, Batam

ChatGPT Impact Analysis on API Testing: A Controlled Experiment

The research demonstrates that students utilizing ChatGPT for API testing achieved higher test scores than those relying on traditional documentation, albeit with longer task completion times. While participants reported enhanced confidence and effectiveness, concerns regarding over-reliance on the tool and limited conceptual understanding were identified. These findings suggest that ChatGPT is a valuable educational resource complemented by independent problem-solving skills.
